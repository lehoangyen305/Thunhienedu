📈 DATA STORAGE
════════════════════════════════════════════════════════════════════════════════

Website Data Flow:
1. User submits form (website/booking)
   ↓
2. Stored in Google Sheets (via Apps Script webhook)
   ↓
3. Admin views in Admin Dashboard (localStorage cache)
   ↓
4. Analytics generated from Google Sheets data

Data Persistence:
- Google Sheets: ✅ Cloud backup (permanent)
- localStorage: ⚠️ Local only (backup recommended)

Export Options:
- Admin → Leads → Export CSV
- Google Sheets → Download CSV/Excel
